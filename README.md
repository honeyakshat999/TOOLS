# TOOLS
This is a script to install tools for recon.
tools:- Subfinder,Dirsearch,arjun,gau,ffuf,aquatone,httpx,gowitness,osmedeus,nuclei etc....

INSTALLTION:

      git clone https://github.com/honeyakshat999/TOOLS.git
      cd TOOLS
      chmod +x script.sh
      ./script.sh 

If not work try:

      sudo ./script.sh;
